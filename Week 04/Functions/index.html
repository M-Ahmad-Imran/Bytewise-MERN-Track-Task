What is Function?

A function in programming is a reusable block of code designed to perform a specific task. Functions can take inputs (known as parameters or arguments), process them, and often return an output. They help to modularize code, making it more readable, maintainable, and reusable.

Key Characteristics of Functions

  1. Name: A function typically has a name that describes what it does. This helps in understanding and using the function.
  2. Parameters: Functions can accept inputs called parameters, which allow them to process data that is passed to them.
  3. Body: The body of a function contains the statements that define the task the function performs.
  4. Return Value: Functions can return a value after performing their task. If no value is returned, the
Function typically returns a special value like undefined in JavaScript.

Syntax of function in JavaScript!

Plain Function:
function name_of_function (parameters) {
//code to be executed
}

Arrow Function:
const name_of_function = (parameters) => {
//code to be executed
}

IIFE
(function() {
//code to be executed
})();
